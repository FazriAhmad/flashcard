# Shiritori Zen - Japanese Word Chain Game

**Shiritori Zen** adalah aplikasi permainan sambung kata Jepang (Shiritori) yang interaktif, dirancang untuk membantu pengguna mempelajari kosakata Bahasa Jepang dengan cara yang menyenangkan dan kompetitif.

## ✨ Fitur Utama
- **Dual Script Support:** Mendukung aksara Hiragana dan Katakana.
- **AI Opponent:** Lawan komputer yang cerdas untuk latihan mandiri.
- **PVP Mode:** Main bersama teman dalam satu perangkat.
- **Text-to-Speech:** Pengucapan kata otomatis dalam Bahasa Jepang.
- **Modern Zen UI:** Antarmuka premium dengan efek Glassmorphism.
- **Survival System:** Sistem nyawa dan timer untuk tantangan ekstra.

## 🛠️ Teknologi
- HTML5 & CSS3 (Vanilla)
- JavaScript (ES6+)
- Web Speech API (untuk fitur suara)

## 🚀 Cara Menjalankan
Cukup buka file `index.html` di browser pilihan Anda.

## 📂 Struktur Proyek
- `index.html`: Struktur utama aplikasi.
- `styles.css`: Desain dan animasi.
- `script.js`: Logika permainan dan integrasi suara.
- `LAPORAN_PROYEK.md`: Laporan detail pengembangan proyek.
